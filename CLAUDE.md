# Hard Rules

- **NEVER create, design, or add a Nodea logo** (icon, SVG, wordmark, symbol, etc.) unless the user explicitly and directly asks for one. This applies everywhere: login pages, navbars, READMEs, favicons, anywhere.

## Project Context

This is the Nodea admin dashboard — the operator interface for the Nodea Software low-code Node.js platform. Built with Next.js 16 (App Router), TypeScript, Tailwind CSS v4, Supabase, Stripe, and the Anthropic Claude API.

Key directories:
- `src/app/` — all Next.js App Router pages, layouts, and API routes
- `supabase/migrations/` — database schema

Official platform: https://nodea-software.com
Docs: https://docs.nodea-software.com
